# Tiny Todo – Early Access

Thank you for testing Tiny Todo! This is a minimal, distraction-free todo list to help you focus on your top tasks.

---

## How to Run

### Windows:
1. Unzip the file if necessary.
2. Double-click `TinyTodo.exe`.
3. Add tasks, check them off, and remove tasks as you complete them.

### Mac:
1. Unzip the file if necessary.
2. Open `TinyTodo.app` (if macOS warns about unidentified developer, right-click → Open).
3. Add tasks, check them off, and remove tasks as you complete them.

---

## Features
- Minimal interface – no clutter, no tabs.
- Floating list – your tasks are always visible.
- Quick add/remove – single click or keystroke to manage tasks.
- Designed for speed – spend your brainpower on tasks, not the list.
